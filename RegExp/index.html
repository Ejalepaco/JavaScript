Demo de Expresiones Regulares en JS

1. Validar Email

Explicación en JS:
const regexEmail = /^[a-zA-Z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,4}$/; const resultado = regexEmail.test(email);

2. Validar Teléfono (España)

Explicación en JS:
const regexTelefono = /^[6|7|9]\d{8}$/; const resultado = regexTelefono.test(telefono);

3. Buscar palabra en texto

Explicación en JS:
const regex = new RegExp(`\\b${palabra}\\b`, 'gi'); const coincidencias = texto.match(regex);
